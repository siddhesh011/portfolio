Projects
Cloud Based Generalized Data Analysis Platform

I developed a versatile cloud-based data analysis platform, streamlining user interaction with a seamless process. Users upload CSV files, stored in an Amazon S3 bucket via API and predefined Lambda functions. Leveraging Amazon Glue for serverless data integration, the platform employs PySpark and Spark ML to conduct descriptive statistics and classification/regression based on the targeted variable. The results are stored back in S3, ensuring accessibility and security. This robust system enhances data analytics capabilities, providing users with actionable insights through a user-friendly interface. The integration of cutting-edge technologies ensures efficiency, scalability, and reliability in data processing and analysis.

NIBRS Data Analysis

The National Incident-Based Reporting System (NIBRS) project entails reporting diverse incident-based statistics to the Polis Center. Data completeness for the years 2019, 2020, and 2021 has been rigorously validated against Census Bureau data. Choropleth maps have been generated to illustrate population densities across respective counties, providing insightful visualizations. To enhance data manipulation, we utilized Python, employing its versatile capabilities for efficient processing. Furthermore, for visualization, Plotly and Bokeh were instrumental, ensuring dynamic and interactive representation of statistical insights. This comprehensive approach not only validates the accuracy of incident data but also presents a clear and engaging portrayal of regional demographics and reporting agencies' statistics.

London Street Cholera Epedemic Dashboard

The project involves the recreation of John Snow's iconic cholera pandemic map using D3.js. Utilizing JSON coordinates and data sourced from a CSV file, the map is dynamically generated to visualize the historical cholera outbreak. The dashboard offers an interactive experience, allowing users to glean valuable insights from the dataset. Through D3.js functionalities, the recreation not only pays homage to Snow's pioneering work but also enhances the accessibility and usability of the cholera data. The integration of interactivity facilitates a comprehensive exploration of the pandemic's geographical and statistical aspects, offering a modern and insightful perspective on historical public health data.

Predictive Analysis On Boston Housing Data

The Boston Housing Price Prediction project underwent exploratory data analysis (EDA) and meticulous data cleaning to enhance dataset quality. Utilizing XGBoost, Random Forest, and Linear Regression models, we aimed to predict housing prices accurately. Each model's unique strengths were harnessed, allowing for a comprehensive evaluation of their performances. Hyperparameter tuning and metric assessments, including Mean Squared Error (MSE), guided the fine-tuning process. This project not only unveils insights into the Boston Housing dataset but also presents a succinct comparative study of three robust models
